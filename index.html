    My final reflection

    Talking about the meesterproef, my goals and minor

1. Get better at communicating

I want to improve my communication skills because I often find myself staying in the background. This time, I want to step forward and share my thoughts more often. I want to contribute more to team projects and ensure my input is heard.

2. Practice Javascript logics

I want to become better in JavaScript and reduce my reliance on Google and ChatGPT for every Javascript problem. By focusing on understanding the logic behind my projects, I want to learn to solve JavaScript problems more on my own. This way it might help me feel more confident and capable as a programmer, so that i can really understand and write my own code.

3. Experiment more

I want to experiment more because I've been told I tend to play it too safe. Fear of underestimating the difficulty of tasks often holds me back, which might cause me to get stuck and make little progress. As a result, I stick mostly to what I already know, trying new things only minimally. Now as goal I want to try to experiment more and try new things.
