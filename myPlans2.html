My goals

My 3 goals for the mastertrial are: Learn to work better in a team, experiment more (with css) and get better with Javascript.

  • Learn to work better in a team

    I want to be able to communicate better within a team and learn to put myself more into the foreground. Give more of my own input into projects.

  • Experiment more (with css)

    I want to dare to experiment more. I do get alot of feedback in the past that i tend to play things a bit safe. By trying out different things i hope i can get more creative and get better in css.

  • Get better with Javascript

    I want to be able to write my own Javascript. Mostly because i tend to Google too much on how to do anything. So want to practice the logic behind my own project more without relying too much on Google.
